# 랜섬웨어 탐지 테스트 샘플

## 개요
이 프로젝트는 랜섬웨어 탐지 시스템을 테스트하기 위한 **무해한** 샘플 파일들을 생성합니다.
실제 악성 행위는 하지 않으며, 정적 분석을 통한 탐지 능력만 검증합니다.

## 생성된 파일

### Sample 1: Ransom Note Pattern
- 파일명: `test_ransom_sample_1.py` / `ransom_test_sample_1.exe`
- 특징:
  - 랜섬 노트 텍스트
  - Bitcoin 지갑 주소
  - 협박 메시지
  - 복구 지시사항

### Sample 2: Crypto Pattern
- 파일명: `test_ransom_sample_2.py` / `ransom_test_sample_2.exe`
- 특징:
  - 암호화 라이브러리 참조
  - 타겟 파일 확장자 목록
  - 암호화 키 패턴
  - 제외 디렉토리 목록

### Sample 3: Network & Persistence Pattern
- 파일명: `test_ransom_sample_3.py` / `ransom_test_sample_3.exe`
- 특징:
  - C&C 서버 주소
  - 레지스트리 키 경로
  - 프로세스 종료 목록
  - 네트워크 포트 정보

## 사용 방법

### 1. Python 스크립트로 실행
```bash
python test_ransom_sample_1.py
python test_ransom_sample_2.py
python test_ransom_sample_3.py
```

### 2. 실행파일 빌드
```bash
# PyInstaller 설치
pip install pyinstaller

# 빌드 스크립트 실행
build_samples.bat
```

빌드된 실행파일은 `dist` 폴더에 생성됩니다.

## ⚠️ 주의사항

1. **격리된 환경에서만 사용**
   - 가상머신 또는 격리된 테스트 환경 권장
   - 프로덕션 환경에서 사용 금지

2. **안티바이러스 경고**
   - 이 파일들은 안티바이러스 솔루션에 의해 탐지될 수 있습니다
   - 이는 정상적인 동작이며, 탐지 시스템이 올바르게 작동하는 것입니다

3. **공개 플랫폼 업로드 금지**
   - VirusTotal 등 공개 플랫폼에 업로드하지 마세요
   - 오탐으로 인해 실제 악성코드로 분류될 수 있습니다

4. **조직 내 사용 시**
   - 보안팀에 사전 통보
   - 승인된 테스트 환경에서만 사용

## 테스트 시나리오

### 정적 분석 테스트
- 파일 헤더 검사
- 문자열 패턴 매칭
- PE 구조 분석
- YARA 룰 적용

### 행위 기반 테스트 (실제 동작 없음)
- 프로세스 모니터링
- 네트워크 트래픽 분석
- 파일 시스템 접근 패턴

## 법적 고지

이 도구는 **교육 및 연구 목적**으로만 제공됩니다.
- 실제 악성 행위를 수행하지 않습니다
- 무단 시스템 테스트는 불법일 수 있습니다
- 사용자는 해당 법률 및 규정을 준수할 책임이 있습니다
